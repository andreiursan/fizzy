# fizzy

A simple fizz-buzz implementation.

## Usage
```
❯ lein repl
fizzy.core=> (map fizz-buzz [1 2 3 5 8 15 23 24 25 28])
   (1 2 "fizz" "buzz" 8 "fizzbuzz" 23 "fizz" "buzz" 28)
fizzy.core=> (fizz-buzz 15)
"fizzbuzz"

```
